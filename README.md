
MacBook Store Landing Page

Project Overview:
This is a responsive landing page built to showcase the MacBook product line. It features interactive animations, 3D MacBook models, and detailed product highlights using React, GSAP, and React Three Fiber. The goal is to demonstrate modern frontend techniques for high-quality web presentation.

Technologies Used:
- React
- React Three Fiber (3D models)
- GSAP for animations
- Tailwind CSS for styling
- React Responsive for media queries
- Zustand for state management

Features:
- Responsive design for desktop and mobile
- 3D MacBook model with rotation
- Scroll-triggered animations for features
- Highlight sections with staggered text animation
- Performance section with animated and positioned images
- Footer with shopping links and legal information
- Optimized for modern browsers

Getting Started:
Clone the repository:
git clone https://github.com/phantekzy/macbook-store.git

Navigate into the project directory:
cd macbook-store

Install dependencies:
npm install

Start the development server:
npm run dev

Build for production:
npm run build

Preview production build:
npm run preview

Project Structure:
- src/components: React components for sections (Performance, Features, Highlights, etc.)
- src/constants: Configuration arrays for navigation, features, images, and videos
- src/models: 3D MacBook model
- src/store: Zustand store for managing video textures and app state
- src/App.jsx: Main React app
- public: Static assets like images, videos, and icons
